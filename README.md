**이미지 색상화(Image Colorization)**
이미지 색상화는 흑백 또는 단색 이미지에 각 픽셀의 절적한 색상을 예측하여 완전한 컬러 이미지로 변환하는 작업
이미지의 구조적 특징과 맥락 정보를 이해하여 자연스러운 색상 결과를 생성하는데 사용
